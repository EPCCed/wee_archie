This is a gentle introduction to the world of parallel computing, covering the
fundamentals of what a parallel computer is, what problems can be solved using
one, and the principles of message passing using MPI.

With all of the introduced concepts, there are accompanying demonstrations that
can be run on Wee Archie, showing how things work using the LED panels.
